## 部署方式

### GitHub Pages

1. 创建GitHub仓库
2. 上传此目录所有文件
3. 在仓库Settings > Pages中启用GitHub Pages
4. 选择main分支作为源

### Vercel

1. 连接GitHub仓库
2. 自动部署
